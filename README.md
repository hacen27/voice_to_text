# Prérequis pour Android

Pour que l'application fonctionne correctement sur Android, assurez-vous de :

- Utiliser un appareil ou un émulateur Android avec Google Play Services.
- Accorder la permission microphone à l'application (demande automatique au lancement, ou à activer dans les paramètres système).
- Avoir une connexion Internet pour de meilleurs résultats de reconnaissance vocale.
- Utiliser un appareil compatible avec la reconnaissance vocale Google (SpeechRecognizer intégré à Android).

**Important** : Si la reconnaissance vocale ne fonctionne pas, vérifiez que l'application a bien la permission micro et que l'appareil supporte la reconnaissance vocale Google.
